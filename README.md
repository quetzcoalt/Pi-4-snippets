# Welcome to my Raspberry vault

Here, I will be trying to put any code I find useful or redundant so I can use it in some other occasion, as well as homeworks from the Paul McWhorter free raspberry pi tutorials. 

## Homeworks:
These Homeworks are proposed solutions to the amazing Paul McWhorter's Raspberry Pi [video tutorial series](https://www.youtube.com/playlist?list=PLGs0VKk2DiYxdMjCJmcP6jt4Yw6OHK85O).

- [Lesson 4 Homework](homework/homework4.py): User-controlled LED blinker.
	- The script asks for:
		- The total number of LED blinks.
		- The amount of delay in seconds between each blink.
- [Lesson 5 Homework](homework/homework5.py): 5-LED Binary Counter (0 to 31) | [video demo](https://youtube.com/shorts/IPr2vUF5JzI)
	- The script increments sequentially from 0 to 31, turning the LEDs on (for 1) and off (for 0). 
- [Lesson 6 Homework](homework/homework6.py): Push button LED blinker.
	- This homework uses different input and output pins to control the LED.
		- When the push button (input) is held down, the LED turns on (output).
		- When released, the LED turns off. 
- [Lesson 7 Homework](homework/homework7.py): LED switching | [video demo](https://www.youtube.com/shorts/-krnzPGeQA8)
	- I use a push button to change the state of the LED to be either ON or OFF.
- [Lesson 9 Homework](homework/homework9.py): Dimmable LED with joystick | [video demo]()
	- I use a joystick to controle the duty cycle of an LED using PWM.
